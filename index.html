FAQs

  • What kind of role does T.P. play in projects?

    T.P. is an system integrator for both hardware and software. We build invisible softwares to connect hardware providers together. We build easy-to-use interfaces to interact with hardwares and other users.

  • What technologies is used in T.P. solutions?

    For hardware, we use high-tech cameras, GPS and BeiDou for outdoor locating, bluetooth for indoor navigation. For software, we use full web technologies which adapts any desktop and mobile devices, Node.js servers which responses realtime between user ends and hardwares.

  • What kind of clients is T.P. focused at?

    T.P. provides solutions for goverments and public service constructions. We started solutions for public security bureaus in Shanghai with their headquater buildings and branches.

  • Can we have a unique software interface?

    Yes, actually we don't have a uniform user interface style. Any solution comes with a full customized front end for specified end users.

  • Can we integrate a solution in securied/local network?

    Yes, we provide systems that can run just with the support of local hardwares and servers. But this is not recommended because system will be isolated and hard-to-maintain, unless you have a very big secured network (like education network or public security network)

  • When should we start a T.P. consultancy?

    Internet of Things should be considered before the first wire is installed into the building, however our solution will come with detail installation plan even in partly-decorated building or already-in-use building / office.
